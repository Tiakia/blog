Http请求之预检请求

2018-03-01 http 阅读次数 0

写了有一段时间的文章了，拿出来再学习学习

使用 options 预检请求目的

  • 可以避免跨域请求对服务器的用户数据产生未预期的影响

不使用 options 预检请求的情况：

  • 使用简单请求的时候，可以不需要进行预检请求，如 GET POST HEAD 方法
  • Content-Type 为 下列三种情况下
    application/x-www-form-urlencoded
    multipart/form-data
    text/plain
  • 请求时不能使用自定义的headers
